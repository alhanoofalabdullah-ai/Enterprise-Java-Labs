# 🚀 Project 15 – Enterprise Fleet Management System

## Overview

This project demonstrates an Enterprise Fleet Management System developed using Java.

The system manages vehicles, drivers, trips, vehicle status, mileage, maintenance records, and fleet operational reporting.

It simulates enterprise fleet systems used by logistics companies, transportation providers, government fleets, and corporate operations teams.

---

# Business Scenario

An organization manages multiple vehicles and drivers.

The fleet operations team needs a centralized system to:

- Register vehicles
- Register drivers
- Assign trips
- Track mileage
- Monitor vehicle availability
- Record maintenance
- Generate fleet reports

---

# Objectives

- Vehicle Management
- Driver Management
- Trip Tracking
- Mileage Monitoring
- Maintenance Tracking
- Fleet Reporting
