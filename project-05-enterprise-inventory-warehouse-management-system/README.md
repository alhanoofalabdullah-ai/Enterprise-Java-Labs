# 🚀 Project 05 – Enterprise Inventory & Warehouse Management System

## Overview

This project demonstrates an Enterprise Inventory and Warehouse Management System developed using Java.

The system manages products, warehouses, stock movements, suppliers, inventory levels, and warehouse reporting through a centralized enterprise platform.

It simulates inventory systems used in ERP platforms, logistics companies, retail operations, and supply chain environments.

---

# Business Scenario

A company manages products across multiple warehouses.

The operations team needs a centralized system to:

- Register products
- Manage warehouse locations
- Track stock quantity
- Record stock-in and stock-out movements
- Monitor low-stock items
- Generate inventory reports
