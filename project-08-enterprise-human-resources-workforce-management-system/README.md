# 🚀 Project 08 – Enterprise Human Resources & Workforce Management System

## Overview

This project demonstrates an Enterprise Human Resources & Workforce Management System developed using Java.

The system manages employee records, attendance, leave requests, performance reviews, workforce statistics, and HR reporting.

The solution simulates enterprise HR platforms such as:

- Oracle HCM
- SAP SuccessFactors
- Workday
- BambooHR
- Zoho People

---

# Business Scenario

A large organization manages hundreds of employees across multiple departments.

The HR department requires a centralized platform to:

- Manage employee records
- Track attendance
- Process leave requests
- Conduct performance reviews
- Monitor workforce metrics
- Generate HR reports
