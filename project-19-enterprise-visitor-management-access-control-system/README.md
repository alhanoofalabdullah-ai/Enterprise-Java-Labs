# 🚀 Project 19 – Enterprise Visitor Management & Access Control System

## Overview

This project demonstrates an Enterprise Visitor Management and Access Control System developed using Java.

The system manages visitor registration, visit requests, approvals, access pass issuance, check-in/check-out activities, visitor status tracking, and security reporting.

It simulates enterprise access control platforms used by corporate offices, government facilities, data centers, hospitals, universities, construction sites, and high-security business environments.

---

# Business Scenario

A corporate facility receives daily visitors including consultants, vendors, contractors, clients, interview candidates, and service providers.

The security and administration teams require a centralized system to:

- Register visitors
- Create visit requests
- Approve visits
- Issue access passes
- Track visitor entry
- Track visitor exit
- Monitor active visitors
- Generate visitor and access reports
- Improve physical security governance

# Core Modules

## Visitor Management

Features:

- Register visitors
- Store visitor information
- Track visitor count
- Maintain visitor records

---

## Visit Request Management

Features:

- Create visit requests
- Assign host department
- Track visit date
- Monitor visit status

Visit statuses include:

- PENDING
- APPROVED
- CHECKED_IN
- COMPLETED

---

## Access Control Management

Features:

- Approve visit requests
- Issue access passes
- Validate access lifecycle
- Track access pass count

---

## Check-In / Check-Out

Features:

- Mark visitor as checked in
- Mark visitor as checked out
- Track active visitors
- Track completed visits

---

## Reporting

Generate:

- Visitor Dashboard
- Access Pass Report
- Pending Visit Report
- Active Visitor Report
- Completed Visit Report
- Security Summary
