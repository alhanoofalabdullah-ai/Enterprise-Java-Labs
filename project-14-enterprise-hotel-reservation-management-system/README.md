# 🚀 Project 14 – Enterprise Hotel Reservation Management System

## Overview

This project demonstrates an Enterprise Hotel Reservation Management System developed using Java.

The system manages guests, rooms, reservations, booking status, cancellations, occupancy tracking, and hotel operational reporting.

It simulates hotel management platforms used by hotels, resorts, hospitality groups, and serviced apartment operators.

---

# Business Scenario

A hotel manages daily reservations across multiple room types.

The operations team needs a centralized system to:

- Register guests
- Manage rooms
- Book reservations
- Cancel reservations
- Track room availability
- Monitor occupancy
- Generate hotel reports

---

# Objectives

- Guest Management
- Room Management
- Reservation Booking
- Reservation Cancellation
- Occupancy Monitoring
- Hotel Reporting
