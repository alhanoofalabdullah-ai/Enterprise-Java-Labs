# 🚀 Project 16 – Enterprise Risk & Compliance Management System

## Overview

This project demonstrates an Enterprise Risk and Compliance Management System developed using Java.

The system manages enterprise risks, internal controls, compliance checks, audit findings, governance status, and executive risk reporting.

It simulates enterprise GRC platforms such as ServiceNow GRC, RSA Archer, MetricStream, LogicGate, and AuditBoard.

---

# Business Scenario

An organization needs a centralized platform to monitor:

- Enterprise Risks
- Compliance Controls
- Audit Findings
- Control Effectiveness
- Compliance Status
- Governance Reports

---

# Objectives

- Risk Registration
- Risk Severity Classification
- Control Management
- Compliance Monitoring
- Audit Finding Tracking
- Governance Reporting
