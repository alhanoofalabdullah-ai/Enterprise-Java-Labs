# 🚀 Project 04 – Enterprise Procurement & Vendor Management System

## Overview

This project demonstrates an Enterprise Procurement & Vendor Management System developed using Java.

The platform manages procurement activities, vendor registration, purchase requests, quotations, purchase orders, and supplier contracts within a centralized environment.

The system simulates enterprise procurement platforms such as:

- SAP Ariba
- Oracle Procurement Cloud
- Coupa
- Ivalua
- Jaggaer

---

# Business Scenario

A large organization purchases products and services from multiple vendors.

The procurement department requires a centralized platform to:

- Register vendors
- Create purchase requests
- Manage quotations
- Generate purchase orders
- Track procurement activities
- Monitor supplier contracts
- Generate procurement reports
