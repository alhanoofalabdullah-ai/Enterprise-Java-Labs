# 🚀 Project 07 – Enterprise Project Management & Task Tracking System

## Overview

This project demonstrates an Enterprise Project Management and Task Tracking System developed using Java.

The system manages projects, tasks, team members, milestones, progress tracking, task status, and project reporting.

It simulates enterprise project management platforms such as Jira, Microsoft Project, Monday.com, Asana, ClickUp, and Primavera-based project management systems.

---

# Business Scenario

An organization manages multiple strategic and operational projects.

The PMO requires a centralized platform to:

- Create projects
- Assign tasks
- Track progress
- Manage milestones
- Monitor team workload
- Generate project reports
- Improve project delivery visibility
