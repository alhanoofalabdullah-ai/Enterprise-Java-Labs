# 🚀 Project 13 – Enterprise Healthcare Appointment Management System

## Overview

This project demonstrates an Enterprise Healthcare Appointment Management System developed using Java.

The system manages patients, doctors, appointments, appointment status, cancellations, and healthcare operational reporting.

It simulates healthcare platforms used by hospitals, clinics, medical centers, and telemedicine providers.

---

# Business Scenario

A healthcare organization receives daily appointment requests from patients.

The administration team needs a centralized system to:

- Register patients
- Register doctors
- Book appointments
- Cancel appointments
- Track appointment status
- Generate healthcare reports

---

# Objectives

- Patient Management

- Appointment Scheduling
- Appointment Cancellation
