# 🚀 Project 20 – Enterprise Project Portfolio Management (PPM) System

## Overview

This project demonstrates an Enterprise Project Portfolio Management (PPM) System developed using Java.

The system manages projects, programs, portfolios, milestones, project risks, project status reporting, and executive portfolio dashboards.

It simulates enterprise project management platforms such as:

- Oracle Primavera P6
- Oracle Primavera Unifier
- Oracle Aconex
- PMWeb
- Microsoft Project Online
- ServiceNow PPM
- Planview
- Clarity PPM

---

# Business Scenario

Large organizations manage dozens of projects across multiple business units.

Executive leadership requires a centralized platform to:

- Register projects
- Group projects into programs
- Track project milestones
- Monitor project risks
- Track portfolio performance
- Generate executive dashboards
- Improve governance and visibility
