# 🚀 Project 06 – Enterprise Helpdesk & IT Service Management System

## Overview

This project demonstrates an Enterprise Helpdesk and IT Service Management System developed using Java.

The system manages support tickets, service requests, agents, priorities, SLA monitoring, ticket escalation, and operational reporting.

It simulates enterprise ITSM platforms such as ServiceNow, Jira Service Management, Freshservice, Zendesk, and ManageEngine ServiceDesk Plus.

---

# Business Scenario

An enterprise IT department receives daily support requests from employees and business units.

The service desk team needs a centralized system to:

- Create support tickets
- Assign tickets to agents
- Track ticket status
- Manage priorities
- Monitor SLA performance
- Escalate unresolved tickets
- Generate IT support reports
