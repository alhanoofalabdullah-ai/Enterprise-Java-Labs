# 🚀 Project 17 – Enterprise Contract Management System

## Overview

This project demonstrates an Enterprise Contract Management System developed using Java.

The system manages vendor contracts, contract values, renewal records, active agreements, and contract governance reporting.

It simulates enterprise contract lifecycle management platforms used in procurement, legal, finance, and vendor governance departments.

---

# Business Scenario

An organization manages multiple supplier and service contracts.

The procurement and legal teams need a centralized system to:

- Register vendors
- Create contracts
- Track contract values
- Monitor active contracts
- Record renewals
- Generate contract reports

---

# Objectives

- Vendor Management
- Contract Registration
- Contract Value Tracking
- Renewal Management
- Contract Governance
- Executive Reporting
